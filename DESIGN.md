# NUCNAS.HUB 网站重新设计方案

参考 MiniMax 官网设计风格完成重新设计。

## 1. 配色方案

采用 MiniMax 风格的深色主题：

| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| `--mm-bg` | `#0A0A0B` | 页面背景（纯黑） |
| `--mm-card` | `#141416` | 卡片背景（深灰） |
| `--mm-border` | `#2A2A2E` | 边框颜色 |
| `--mm-hover` | `#1E1E22` | 悬停状态 |
| `--mm-accent` | `#8B5CF6` | 强调色（紫色） |
| `--mm-gradient-start` | `#6366F1` | 渐变起始（蓝） |
| `--mm-gradient-end` | `#8B5CF6` | 渐变结束（紫） |
| `--mm-text` | `#FAFAFA` | 主文字 |
| `--mm-text-secondary` | `#A1A1AA` | 次要文字 |

## 2. 字体方案

- **中文**：`Noto Sans SC` + `Inter`
- **英文/数字**：`Inter`
- **代码/价格**：`JetBrains Mono`

字号层级：
- 标题：36px / 56px (Hero)
- 正文：15px
- 辅助：13px / 12px

## 3. 首页布局

- **Hero 区域**：大字标题 + 渐变文字强调 + 搜索热词标签
- **栏目卡片**：4个大卡片，使用图标 + 渐变边框
- **热门设备**：4列卡片网格，展示评分和价格
- **最新文章**：简洁列表，包含日期和标签

## 4. 列表页布局

- **筛选栏**：品牌/类型/价格下拉 + 标签筛选
- **卡片网格**：3列 (桌面) / 2列 (平板) / 1列 (手机)
- **分页器**：简洁风格，当前页高亮

## 5. 详情页布局

- **标题区**：大标题 + 标签 + 阅读时间/日期
- **内容区**：优化的阅读体验，最大宽度舒适行宽
- **评论区**：包含评论表单 + 示例评论
- **相关文章**：底部3卡片推荐

## 6. 搜索功能

- **导航栏搜索框**：按回车触发搜索
- **搜索热词**：首页显示热门搜索标签
- **搜索结果页**：分类筛选 + 相关性排序

## 7. 评论互动

- **评论表单**：昵称输入 + 评论文本框 + 提交按钮
- **评论列表**：头像 + 作者 + 时间 + 内容 + 点赞/回复
- **首页评论区**：展示热门评论示例

## 已修改的文件

1. `tailwind.config.js` - 更新配色方案
2. `layouts/_default/baseof.html` - 基础模板 + 导航 + 搜索框
3. `layouts/index.html` - 首页布局 + 评论区
4. `layouts/_default/list.html` - 通用列表页
5. `layouts/_default/single.html` - 详情页 + 评论 + 相关文章
6. `layouts/_default/search.html` - 搜索结果页
7. `layouts/hardware/list.html` - 硬件列表页
8. `layouts/nas/list.html` - NAS列表页
9. `assets/css/main.css` - 完整样式表

## 设计特点总结

✅ **深色主题**：纯黑背景 + 深灰卡片，简洁现代
✅ **渐变强调**：紫蓝色渐变用于按钮、评分、文字强调
✅ **大面积留白**：卡片内边距充足，视觉呼吸感强
✅ **卡片展示**：圆角16-20px，微妙的边框和阴影
✅ **层次清晰**：标题-正文-辅助信息层级分明
✅ **交互反馈**：悬停效果、点击动画
