## 一、平台定位与用户画像

### 1.1 核心定位
- **Slogan**：专注 NAS、迷你主机(Mini PC)、本地AI 的测评与方案推荐
- **品牌调性**：硬核、专业、实用、不恰饭
- **三大支柱**：
  - 🖥️ **NAS** - 私有云存储方案
  - 💻 **迷你主机** - Mini PC 选购指南  
  - 🤖 **AI** - 本地大模型部署

| 平台 | 用户特征 | 痛点 | 需求 |

## 二、网站内容规划 (nucnas.top)

### 2.1 栏目结构

| 栏目 | 路径 | 内容定位 | 更新频率 |
|------|------|----------|----------|
| 硬件测评 | /hardware/ | 迷你主机横向评测 | 每周2篇 |
| NAS学院 | /nas/ | 系统安装、配置教程 | 每周2篇 |
| AI | /ai/ | 本地大模型部署 | 每周1篇 |
| 计算器 | /calculator/ | 性能功耗对比工具 | 持续优化 |

### 2.2 内容矩阵

#### 【横向评测】（30%）
```
格式：X款Y元价位产品对比

必备要素：
- 参与产品参数表
- 性能测试数据（CPU/GPU/内存/磁盘）
- 噪音/温度实测
- 优缺点总结
- 购买建议

示例：
- 《2026年千元NAS对比：绿联/极空间/群晖怎么选》
- 《5款高性能迷你主机横评：AMD vs Intel》
- 《Intel N100/N150/N305 性能对比表》
- 《本地AI部署：Ollama vs LM Studio 对比》
```

#### 【单品测评】（25%）
```
格式：深度测评 + 优缺点 + 适用场景

必备要素：
- 产品基本参数
- 外观与接口
- 性能测试（跑分、游戏实测）
- 温度与噪音
- 优缺点总结
- 适合人群
- 竞品对比

示例：
- 《极空间Z4Pro测评：可能是最适合小白的NAS》
- 《Mac mini M4 深度测评：做NAS香不香？》
```

#### 【教程】（25%）
```
格式：手把手指南

必备要素：
- 准备工作（硬件/系统镜像）
- 步骤截图
- 常见问题FAQ
- 进阶优化（可选）

示例：
- 《TrueNAS Scale 安装指南：从U盘到开机》
- 《Docker 配置家中软路由》
- 《NAS硬盘选购指南：CMR vs SMR》
```

#### 【话题/观点】（20%）
```
格式：观点文 + 讨论引导

示例：
- 《NAS到底值不值？算一笔账》
- 《为什么我不推荐买群晖》
- 《迷你主机能否替代台式机？》
```

### 2.3 SEO 优化策略

| 品类词 | NAS推荐、迷你主机 | 标题、H1 |

## 三、头条内容规划

### 3.1 内容类型

| 类型 | 占比 | 标题风格 | 字数 |
|------|------|----------|------|
| 热点蹭流 | 40% | 「刚刚发布」「全网首发」 | 500-800 |
| 性价比推荐 | 30% | 「百元方案」「省钱必看」 | 600-1000 |
| 行业观察 | 20% | 「深度分析」「行业趋势」 | 800-1200 |
| 互动话题 | 10% | 「你怎么看」「评论区聊聊」 | 300-500 |

### 3.2 爆款标题模型

#### 数字吸睛型
- 「省300块！NAS选购避坑指南」
- 「5款热门迷你主机对比，这款性价比最高」
- 「3分钟看懂NAS怎么选」

#### 痛点解决型
- 「NAS小白必看：这是我踩过的坑」
- 「别再乱买NAS了，看完这篇再决定」
- 「入手NAS前，这5点必须知道」

#### 悬念型
- 「为什么我不推荐买群晖？真相来了」
- 「NAS真的值得买吗？用了3年说说感受」
- 「这款NAS凭啥卖断货？实测告诉你」

### 3.3 写作模板

#### 模板一：对比推荐
```
【开头】直接痛点：想买NAS但不知道选哪个？
【中间】3款热门产品参数对比 + 优缺点
【结尾】根据需求给出推荐 + 引导评论
```

#### 模板二：避坑指南
```
【开头】NAS选购5大误区，你中招了吗？
【中间】逐条分析 + 避坑建议
【结尾】评论区聊聊你踩过什么坑
```

#### 模板三：热点蹭流
```
【开头】Intel N150刚刚发布，看看值不值
【中间】对比上代N100，提升多少？
【结尾】建议观望还是入手？
```

### 3.4 发布规范

| 项目 | 要求 |
|------|------|
| 封面 | 三图或单图，文字居中 |
| 标题 | 18-25字，含关键词 |
| 正文 | 500-1200字，分段清晰 |
| 话题 | 3-5个相关话题 |
| 互动 | 引导评论、点赞 |
| 收益 | 必须勾选「投放广告赚收益」 |

## 四、小红书内容规划

### 4.1 内容类型

| 类型 | 占比 | 封面风格 | 关键词 |
|------|------|----------|--------|
| 方案推荐 | 30% | 对比图/清单 | 小白、入门、指南 |
| 避坑指南 | 20% | 警示风/对比 | 避坑、别买、后悔 |
| 性价比 | 20% | 省钱风/清单 | 千元、平价、DIY |
| **桌搭展示** | **20%** | **实拍图/氛围感** | **桌面搭配、数码桌搭、Minimalist** |
| 场景展示 | 10% | 实拍图/治愈 | 书房、角落、整洁 |

### 4.2 爆款标题模型

#### 反差型
- 「10㎡小房间搞定家庭影院，绝了！」

#### 痛点型
- 「求求你们别这样选NAS了，全是坑！」

#### 保姆型
- 「全网最全！3步搞定NAS，小白也能学会」

#### 情绪型
- 「一个人住也要热气腾腾，我的书房角落」

#### 桌搭型（新增）
- 「强迫症患者桌面分享｜极简数码桌搭」
- 「千元打造高效工作台｜迷你主机+显示器」
- 「我的Mini PC桌搭｜简约不简单」

### 4.3 封面设计规范

| 场景 | 建议 |
|------|------|
| 对比 | 左边错误方案 / 右边正确方案 |
| 清单 | 背景纯色 + 大字 + emoji |
| 避坑 | ❌标志 + 警示色 |
| 实拍 | 场景图 + 文字贴纸 |
| **桌搭** | **整体场景 + 局部特写 + 产品清单** |

### 4.4 正文结构

```
【黄金3行】前3行必须包含核心价值
❶ 问题/痛点引入
❷ 解决方案（核心干货）
❸ 行动引导

【正文】清单化呈现
✅ 优点1
✅ 优点2
❌ 缺点/注意事项

【互动引导】
「有问题评论区见」
「链接在评论区」
「点赞收藏防丢失」
```

### 4.5 标签策略

```
2个大流量词 + 2个精准细分词 + 1个热点词

示例：
#NAS #家庭存储 #私有云 #小米NAS #2026
```

### 4.6 发布规范

| 项目 | 要求 |
|------|------|
| 频率 | 每天1-2篇 |
| 图片 | 9张以内，文字简洁 |
| 话题 | 3-5个 |
| 互动 | 回复评论，引导关注 |
