# 数据分析监控

监控网站数据。

## 功能

- 统计访问量 (PV/UV)
- 热门文章排行
- 流量来源分析
- 关键词排名
- 生成数据报告

## 使用方法

```bash
# 查看今日数据
analytics --today

# 查看周报
analytics --weekly

# 热门文章
analytics --top-articles
```

## 报告输出

自动生成 JSON/HTML 格式报告，方便查看和分享。
