# SEO 优化器

自动优化网站 SEO。

## 功能

- 生成/更新 sitemap.xml
- 检查并修复死链
- 优化 meta 标签
- 生成 robots.txt
- 提交搜索引擎收录

## 使用方法

```bash
# 完整 SEO 优化
seo-optimizer --full

# 只生成 sitemap
seo-optimizer --sitemap

# 检查死链
seo-optimizer --check-links
```
