# 小红书发布助手

自动生成并发布小红书种草笔记。

## 使用方法

```powershell
# 生成小红书内容
.\xiaohongshu-post.ps1 -Article "apple-mac-mini-m4"
```

## 内容模板

```
【标题】

正文内容...

#标签1 #标签2 #标签3 #标签4 #标签5
```

## 热门标签

- 数码
- 科技
- 测评
- NAS
- 迷你主机
- 数码好物
- 科技测评

## 风格指南

- 标题要吸引眼球
- 内容要真实、有用
- 加上表情符号
- 结尾引导互动
