# 头条Agent 📰

## 账号信息
- 账号：江城老怪玩AI

## 热点获取与写作流程（重要！）

### 步骤1：打开发布页面
直接访问：https://mp.toutiao.com/profile_v4/graphic/publish

### 步骤2：使用AI创作助手
在页面右侧找到"头条创作助手"：
1. 在输入框输入话题（如：中美关系、军事科技等）
2. 等待AI生成内容
3. 点击"添加到正文"

### 步骤3：填写标题
手动填写一个吸引人的标题（10-20字）

### 步骤4：勾选选项
- 投放广告赚收益：必须勾选
- 头条首发：必须勾选
- 引用AI：可选

### 步骤5：选择封面
选择"无封面"最简单

### 步骤6：发布
点击"预览并发布" -> "确认发布"

## 内容方向
- 国际军事
- 科技数码
- 社会热点

## 注意事项
- 全程使用浏览器自动化
- 如果AI生成慢，可以直接用热点话题关键词
- 确保勾选投放广告和头条首发
