## 方案 A：全自动发布（推荐）

### 前置条件
1. ✅ Chrome 浏览器已安装
2. ✅ 已登录头条号 (mp.toutiao.com)
3. ✅ Node.js 已安装
4. ✅ Playwright 已安装

### 安装依赖
```bash
cd C:\Users\nczix\.openclaw\workspace
npm install playwright
```

### 执行发布
```bash
node scripts/toutiao_publisher.js articles/2026-03-08-01.md
```

### 自动化流程
1. 启动 Chrome（使用现有登录状态）
2. 打开发布页面
3. 填写标题
4. 填写正文
5. ✅ 自动勾选"投放广告赚收益"
6. ✅ 自动勾选"头条首发"
7. 点击发布

---

## 方案 B：半自动发布（当前）

### 步骤
1. 打开头条号后台：https://mp.toutiao.com/
2. 点击"发布文章"
3. 从 `agents/toutiao/articles/` 复制文章
4. 粘贴标题和正文
5. **手动勾选**：
   - ✅ 投放广告赚收益
   - ✅ 头条首发
6. 选择封面图（单图/三图/无图）
7. 点击"发布"

---

## 常见问题

### Q1: 提示"未登录"
**解决**：先用 Chrome 登录头条号，关闭浏览器后再运行脚本

### Q2: "投放广告"选项找不到
**解决**：可能是页面结构变化，需要更新选择器配置

### Q3: 发布失败
**解决**：
1. 检查网络连接
2. 检查账号状态（是否被限流）
3. 检查文章内容（是否违规）
