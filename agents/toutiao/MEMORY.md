# MEMORY.md - 头条运营记忆

## 账号信息
- **平台**: 今日头条
- **账号名**: 江城老怪玩 AI
- **内容方向**: 国际军事 / 科技数码 / 社会热点

## 运营策略

### 选题来源
1. **热点猎手**: 每小时扫描头条热榜、微博热搜、百度热榜
2. **对标账号**: 监控 5-10 个同领域大 V 的爆款
3. **粉丝反馈**: 评论区和私信的高频问题

### 爆款公式
- **标题**: 数字 + 痛点/悬念 + 情绪词 (例："3 个信号表明..."、"刚刚，XXX 曝光！")
- **开头**: 3 句话内点明核心信息
- **正文**: 分点论述，每段不超过 5 行
- **结尾**: 总结 + 互动引导 ("你怎么看？评论区聊聊")

### 发布规则
- **频率**: 每天 3-5 篇
- **时间**: 早 7-9 点 / 午 12-14 点 / 晚 19-22 点
- **必勾选**: ✅ 投放广告赚收益 ✅ 头条首发
- **封面**: 单图优先，清晰有冲击力

## 变现路径
1. **广告分成**: 基础收入，靠阅读量
2. **专栏付费**: 深度内容变现
3. **带货**: 数码产品、书籍等
4. **引流**: 导流到私域/其他平台

## 数据目标
- **短期**: 日均阅读 10 万+
- **中期**: 粉丝破 5 万
- **长期**: 月收入 5000+
