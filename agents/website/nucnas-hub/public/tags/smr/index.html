    CMR vs SMR：NAS硬盘选购必懂的坑

    前言

    买NAS硬盘，你一定听说过CMR和SMR这两个术语。它们直接关系到硬盘的性能和寿命，但很多新手用户并不了解其中的区别。本文将用最通俗的语言，帮你彻底搞懂这个NAS硬盘选购中最大的"坑"。

    一、什么是CMR？

    1.1 CMR技术原理

    CMR（Conventional Magnetic Recording，传统磁记录），也称为垂直记录（PMR）。

    • 写入数据时，磁头在盘片上创建独立的磁轨
    • 磁轨之间有明确的间隙，互不干扰
    • 数据可以随时重写，不影响相邻磁轨

    1.2 CMR的特点

    ✅ 优点：

    • 随机写入性能优秀
    • 长时间使用性能稳定
    • 适合NAS/RAID环境
    • 寿命更长

    ❌ 缺点：

    • 存储密度相对较低
    • 单位容量成本较高

    二、什么是SMR？

    2.1 SMR技术原理

    SMR（Shingled Magnetic Recording，叠瓦式磁记录）。

    • 磁轨像屋顶瓦片一样重叠排列
    • 写入新数据时，可能需要重写相邻磁轨
    • 提高了存储密度，降低了成本

    2.2 SMR的特点

    ✅ 优点：

    • 存储密度高
    • 单位容量成本低
    • 适合纯存储场景

    ❌ 缺点：

    • 随机写入性能差
    • 长时间使用性能下降明显
    • 不适合NAS/RAID环境
    • 故障风险较高
