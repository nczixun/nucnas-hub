很多玩家想在家用服务器上跑多个服务，Proxmox VE（简称 PVE）是目前最流行的方案。今天手把手教你安装。

## 准备工作

1. 一台 x86 主机（Intel/AMD 均可）
2. U 盘一个（≥ 8GB）
3. PVE 镜像（官网下载 iso）

## 安装步骤

### 1. 制作启动盘

用 Rufus 或 Ventoy 将 PVE 镜像写入 U 盘。建议使用 Ventoy，支持多镜像。

### 2. 设置 BIOS

开机进 BIOS，关闭 Secure Boot，设置为 U 盘启动。

### 3. 开始安装

选择 "Install Proxmox VE"，一路 Next。需要注意：

- **硬盘分区**：如果只装 PVE，可以用整个磁盘；如果要跑 NAS，建议预留空间
- **时区**：选择 Asia/Shanghai
- **网络**：建议配置静态 IP，方便后续访问

### 4. 初始化配置

安装完成后，重启进入系统。浏览器访问 `https://你的IP:8006`，用 root 账号登录。

首次登录会提示创建 PVE 订阅，可以先跳过。

## 常见问题

**Q: 安装后无法启动？**
检查 BIOS 中的 CSM 和 UEFI 设置，确保与启动模式匹配。

**Q: 网页打不开？**
确认防火墙是否放行 8006 端口。

## 下一步

安装完成后，建议：
1. 更新到最新版本
2. 创建第一个虚拟机（建议先跑一个轻量 Linux 熟悉界面）
3. 配置 RAID 或存储池

有问题欢迎评论区交流！
