# Ollama 模型量化指南：教你用 4GB 显存运行 70B 大模型

很多朋友显卡显存不够，又想跑大模型，怎么办？答案就是**量化**。今天聊聊 Ollama 的量化模型怎么选。

## 什么是量化？

简单说就是将**FP16（16位浮点）** 压缩到**Q4/Q5（4-5位整数）**，显存占用直接砍半甚至更多。

## 量化等级一览

| 量化级别 | 精度 | 70B 模型显存 | 推荐场景 |
|----------|------|--------------|----------|
| FP16 | 16bit | ~140GB | 3090/4090 旗舰 |
| Q4_K_M | 4bit | ~40GB | 24GB 显存 |
| Q5_K_S | 5bit | ~50GB | 24GB 显存 |
| Q8_0 | 8bit | ~70GB | 3090 显存 |

## 如何选择量化模型？

1. **24GB 显存**：选择 Q4_K_M 或 Q5_K_S
2. **16GB 显存**：选择 Q3_K_M 或 Q4_0
3. **8GB 显存**：选择 Q2_K 或 Q3_0

## Ollama 量化命令

```bash
# 查看可用量化级别
ollama list

# 拉取量化模型
ollama run llama2:70b-q4_k_m
```
