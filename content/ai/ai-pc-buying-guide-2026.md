# AI PC 选购指南：2026年最适合大模型的电脑

## 什么是 AI PC？

AI PC 是专门为本地运行 AI 大模型而配置的电脑，需要强大的 CPU 和 GPU 性能。

## 核心配置要求

### 最低配置（运行 7B 模型）

| 组件 | 要求 |
|------|------|
| CPU | Intel i5 / AMD Ryzen 5 |
| 内存 | 16GB |
| 显卡 | GTX 1660 Ti / RTX 3060 |
| 存储 | 512GB SSD |

### 推荐配置（运行 13B 模型）

| 组件 | 推荐 |
|------|------|
| CPU | Intel i7 / AMD Ryzen 7 |
| 内存 | 32GB |
| 显卡 | RTX 4070 12GB |
| 存储 | 1TB NVMe SSD |

### 高端配置（运行 70B 模型）

| 组件 | 配置 |
|------|------|
| CPU | Intel i9 / AMD Ryzen 9 |
| 内存 | 128GB |
| 显卡 | RTX 4090 24GB × 2 |
| 存储 | 2TB NVMe SSD |

### AMD 显卡

AMD 显卡在 AI 推理方面较弱，推荐用于游戏，不推荐用于 AI。
