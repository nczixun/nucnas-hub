# 零刻 EQ59 Pro 评测：千元级入门NAS首选

零刻 EQ59 Pro 是一款定位于入门级市场的迷你主机，搭载 Intel N5105 处理器，非常适合作为软路由或轻量级 NAS 使用。

## 核心配置

| 组件 | 规格 |
|------|------|
| 处理器 | Intel Celeron N5105 |
| 核心数 | 4 核 4 线程 |
| 基础频率 | 2.0 GHz |
| 睿频 | 2.9 GHz |
| 内存 | 16GB DDR4 |
| 存储 | 512GB SSD |

## 适用场景

### ✅ 适合

- 软路由（OpenWrt、爱快）
-轻量级 NAS
- 下载机
- Home Assistant 家庭智能中枢
- Docker 运行轻量容器

### ❌ 不适合

- 4K 视频转码
- 虚拟机多开
- 大型 AI 模型运行
