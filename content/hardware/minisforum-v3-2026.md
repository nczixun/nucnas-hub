# Minisforum V3深度评测：AMD 8840U加持，三合一平板能否取代传统笔记本？

**测评日期：** 2026年3月

## 一、 前言

Minisforum V3是一款极具创新性的产品——它既可以作为Windows平板使用，又可以连接键盘底座变成笔记本，还能通过视频输出作为外接显示器。这种"三合一"设计理念，在Windows阵营中相当罕见。

今天我们就来详细评测这款产品。

## 二、 外观与设计

### 2.1 整机设计

**尺寸与重量：**
- 厚度：9.8mm
- 重量：约946g（不含键盘）

**材质：**
- 全金属机身
- 磨砂工艺处理

### 2.2 屏幕

| 参数 | 规格 |
| :--- | :--- |
| 尺寸 | 13.3英寸 |
| 分辨率 | 2560×1600（2.5K） |
| 刷新率 | 165Hz |
| 亮度 | 500nit |
| 色域 | 100% DCI-P3 |

### 2.2 接口

| 位置 | 接口 |
| :--- | :--- |
| 左侧 | USB4 40Gbps、全功能USB-C、3.5mm音频 |
| 右侧 | 音量键、电源键 |
| 底部 | 键盘磁吸接口、Surface风格充电接口 |

**注意**：无USB-A接口，需要转接。

## 三、 核心配置

### 3.1 处理器

| 型号 | 核心/线程 | 最高频率 | TDP |
| :--- | :--- | :--- | :--- |
| **8840U** | 8C/16T | 5.1GHz | 28W |
| 8640U | 6C/12T | 4.9GHz | 28W |

### 3.2 存储与内存

- **内存**：16GB/32GB LPDDR5-7500（板载）
- **存储**：M.2 2280 PCIe 4.0插槽，最高4TB

### 3.3 续航

- 电池容量：50Wh
- 官方续航：约8小时
- 充电：65W PD快充

## 五、 三种模式体验

### 5.1 平板模式

- 适合手写笔记、绘图
- 支持4096级压感手写笔（需另购）
- 165Hz高刷，触控流畅

### 5.2 笔记本模式

- 磁吸键盘设计
- 键盘手感类似Surface Pro
- 触控板支持多指手势

### 5.3 显示器模式

- 可作为外接显示器使用
- 适合Switch等设备投屏
- 2.5K 165Hz，素质不错

## 六、 散热与噪音

### 烤机测试

| 模式 | CPU温度 | 噪音 |
| :--- | :--- | :--- |
| 平衡模式 | 75℃ | 38dB |
| 性能模式 | 85℃ | 45dB |

高负载下风扇噪音偏大，这是小尺寸高性能设备的通病。

## 七、 总结

**优点：**
- ✅ 三合一设计创新
- ✅ 8840U性能强劲
- ✅ 2.5K 165Hz屏幕素质高
- ✅ 机身轻薄便携

**缺点：**
- ❌ 无USB-A接口
- ❌ 高负载噪音明显
- ❌ 散热受限

**购买建议：**
- **适合**：需要多场景切换、追求便携的专业用户
- **不适合**：需要长时间高性能输出的用户
