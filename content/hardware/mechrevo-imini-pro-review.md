# 机械革命imini Pro评测：Ultra 5+玻璃顶盖，游戏风格桌面主机

提到迷你主机，很多人第一印象是"商务范"、"方正低调"。但**机械革命imini Pro**要打破这个印象——玻璃顶盖、RGB灯效、炫酷游戏风，这款主机要把迷你主机带入"潮玩"时代。

## 核心配置

| 配置项 | 参数 |
|--------|------|
| 处理器 | Intel Core Ultra 5 125H (14核18线程) |
| 核显 | Intel Arc Graphics (7核心) |
| 内存 | DDR5 5600MHz (双通道) |
| 存储 | M.2 2280 NVMe SSD |
| 网口 | 2.5G × 1 |
| 功耗 | 整机约35-65W |

## 外观设计

imini Pro最大的亮点是**玻璃顶盖**设计。顶部采用钢化玻璃材质，可以看到内部散热器和RGB灯效，科技感满满。

机身尺寸约为**1.2L**，采用**游戏化设计语言**：
- 棱角分明的外壳
- RGB灯条 (支持1680万色调节)
- 玻璃顶盖下的散热器和风扇

机身接口：
- USB-C (Thunderbolt 4) × 1
- USB 3.2 × 3
- HDMI 2.1 × 1
- DP 1.4 × 1
- 2.5G网口 × 1
- 3.5mm音频接口

### RGB灯效

通过内置软件可以自定义灯效：
- 静态单色
- 呼吸灯
- 彩虹循环
- 音乐律动

游戏时配合键盘鼠标，营造沉浸式氛围。

## 散热设计

imini Pro采用**单风扇+单热管**散热：

- 日常使用：风扇停转 (被动散热)
- 中度负载：约30dB
- 满载：约38dB

玻璃顶盖虽然好看，但对散热有一定影响。长时间高负载使用建议搭配散热底座。
