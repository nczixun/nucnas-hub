# 家庭数据中心新选择——Aoostar WTR Pro 评测：四盘位NAS主机也能性能强劲

**测评日期：** 2026年3月1日  
**测评型号：** Aoostar WTR Pro（AMD Ryzen 7 5800U / 16GB+无盘版）


## 一、 前言

对于追求大容量存储和All-in-One功能的用户来说，传统NAS性能往往不够强劲，而高性能迷你主机又缺乏足够的硬盘位。**Aoostar WTR Pro**的出现，完美填补了这一空白——它搭载AMD Ryzen 7 5800U处理器，提供**四个3.5英寸硬盘位**，同时还支持虚拟机、Docker等高级功能，堪称“全能家庭数据中心”。

今天我们就来详细评测这款产品，看看它是否值得入手。


## 二、 外观与设计：扎实做工，NAS该有的样子

**体积与设计**
WTR Pro的机身尺寸为**240×200×150mm**，体积约**7.2升**，比传统ITX机箱略小，但比普通迷你主机大了一圈。全身采用**金属材质**，质感扎实，黑色外观简约大气，非常适合放在家中的弱电箱或储物间。

**前置接口**
- 电源键
- USB 3.0 × 2
- 3.5mm音频接口
- 硬盘仓指示灯

**后置接口**
- DC电源输入（12V/5A）
- HDMI 2.0
- USB 3.0 × 2
- USB 2.0 × 2
- 2.5G RJ45网口 × 2
- PCIe x4扩展槽


## 三、 硬件配置：5800U加持，性能不再是瓶颈

| 组件 | 规格 |
| :--- | :--- |
| **处理器** | AMD Ryzen 7 5800U（8核16线程，基础1.9GHz/加速4.4GHz） |
| **集成显卡** | AMD Radeon Vega 8（8核，2000MHz） |
| **内存** | 2×SO-DIMM，支持最高64GB DDR4-3200MHz |
| **存储** | 1×M.2 2280 NVMe插槽（支持PCIe 3.0×4） |
| **硬盘位** | 4×3.5英寸SATA硬盘位 |
| **网络** | 2×2.5G RJ45（Realtek RTL8125） |

**性能表现**
5800U虽然在绝对性能上不如最新的7000系列，但对于NAS应用来说已经**性能过剩**。实测：
- CPU-Z单核得分：580分，多核得分：4500分
- 核显Vega 8可以轻松硬解4K视频，支持Jellyfin/Plex转码
- 虚拟机运行Windows Server或Ubuntu完全没有压力


## 四、 四盘位存储：真正的NAS体验

**硬盘位设计**
WTR Pro提供**四个3.5英寸硬盘位**，支持热插拔，单盘最高支持**18TB**机械硬盘，总容量可达**72TB**。对于普通家庭用户来说，这个容量足以存储几年的照片、视频和文件。

**RAID支持**
通过系统设置，可以组建：
- RAID 0/1/5/6/10
- JBOD（线性扩展）
- Basic（独立磁盘）

**散热设计**
机身内置**两个8cm静音风扇**，位于硬盘仓背部，可以有效带走硬盘热量。日常运行时噪音约**30dB**，非常安静；满载运行时噪音约**45dB**，在可接受范围内。


## 五、 系统与功能：All-in-One的理想选择

WTR Pro支持多种NAS系统：

**官方OS**
- 基于OpenWrt定制的Aoostar OS
- 图形化界面，易于上手
- 支持Docker、虚拟机

**自行安装**
- TrueNAS Scale
- Unraid
- Proxmox VE
- Ubuntu/Debian（作为NAS使用）

**核心功能**
- 文件存储与共享（SMB/NFS/AFP）
- Docker容器（Home Assistant、Jellyfin、Nextcloud等）
- 虚拟机（Windows、Linux）
- 照片备份（Time Machine、Sync+）
- 下载机（Transmission、Qbittorrent）


## 六、 适合人群与选购建议

**适合人群：**
- 需要大容量存储的家庭用户
- 想搭建All-in-One的玩家
- 需要稳定运行的Homelab用户
- 想入门NAS但不想买成品群晖/威联通的用户

**不适合人群：**
- 对噪音极度敏感的用户
- 需要万兆网络的极客用户
- 追求极致小巧的用户

**选购建议：**
- 建议购买准系统版本，自行购买内存和硬盘，性价比更高
- 内存建议32GB以上，方便运行虚拟机和Docker
- 硬盘建议选择NAS专用盘（如希捷酷狼、西数红盘）


## 七、 总结

Aoostar WTR Pro是一款**定位清晰、性价比极高**的四盘位NAS主机。它没有追求极致的性能或接口，而是在**存储容量、系统扩展性和价格**之间取得了很好的平衡。对于想搭建家庭数据中心、All-in-One服务器或简单NAS的用户来说，这是一个**值得考虑的选择**。
