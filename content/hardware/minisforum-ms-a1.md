# 桌面级性能的“变形金刚”——铭凡 Minisforum MS-A1 评测：AM5插槽可换CPU

**测评日期：** 2026年3月1日  
**测评型号：** 铭凡 Minisforum MS-A1（AMD Ryzen 7 8700G / 32GB+1TB版）


## 一、 前言

在迷你主机领域，**“可升级”**一直是个伪命题——大多数产品都是板载CPU，用户无法自行升级。但今天这台**Minisforum MS-A1**彻底改变了这一局面，它是业界首批采用**AM5插槽**的迷你主机，用户可以像组装台式机一样**自行更换CPU**！

这意味着什么？你现在买一台入门级的5600G，日后可以升级到最强的7950X3D。一台迷你主机，可以用上十年甚至更久。让我们一起来看看这款“未来产品”。


## 二、 外观与设计：服务器级做工，桌面级扩展

**体积与设计**
MS-A1的机身尺寸为**170×170×50mm**，体积约**1.4升**，略大于普通NUC，但依然非常紧凑。全金属机身，质感扎实，顶部采用**蜂窝状散热孔**设计，既美观又利于散热。

**接口配置：应有尽有**

| 位置 | 接口类型 | 规格与用途 |
| :--- | :--- | :--- |
| **前置** | 电源键、USB4（40Gbps）、USB 3.2 Gen2、3.5mm音频 | 高速传输，便携设备 |
| **后置** | 2×HDMI 2.1、2×2.5G RJ45、USB 3.2 Gen2×3、USB 2.0×2、DC 19V | 多屏办公，网口聚合 |

**最关键的设计：AM5插槽**
机身内部提供一个**标准AM5插槽**，支持AMD Ryzen 7000/8000系列桌面级处理器。这意味着你可以安装：
- Ryzen 5 5600G（入门选择，约400元）
- Ryzen 7 8700G（核显怪兽，约1200元）
- Ryzen 9 7950X（极致性能，约2500元）
- 未来zen5/zen6架构的CPU


## 三、 硬件配置：8700G核显，性能巅峰

| 组件 | 规格 |
| :--- | :--- |
| **处理器** | AMD Ryzen 7 8700G（8核16线程，基础4.2GHz/加速5.1GHz） |
| **集成显卡** | AMD Radeon 780M（12核，2900MHz） |
| **内存** | 2×SO-DIMM，支持最高96GB DDR5-5200MHz |
| **存储** | 2×M.2 2280 NVMe插槽（均支持PCIe 4.0×4） |
| **网络** | 2×2.5G RJ45（Realtek RTL8125） |

**性能表现**
8700G是目前最强的APU，实测：
- CPU-Z单核得分：700分，多核得分：7500分
- **Radeon 780M核显**：Time Spy得分约3200分，可以流畅运行《原神》《崩坏：星穹铁道》等游戏
- 核显支持**硬件光追**，这是历史首次！
- 32GB DDR5双通道下，核显性能接近GTX 1050 Ti


## 四、 散热系统：散热压得住吗？

**散热器设计**
MS-A1配备一个**大型VC均热板散热器**，风扇为**12cm静音风扇**。虽然8700G的TDP只有65W，但考虑到迷你主机有限的散热空间，这个散热设计还是很有必要的。

**散热测试**
- 待机温度：35°C
- 单烤CPU：75°C（风扇转速40%，噪音28dB）
- 双烤CPU+GPU：85°C（风扇转速70%，噪音42dB）
- 游戏温度：70-80°C

**散热评价**
整体散热表现良好，即使长时间满载也不会出现严重降频。噪音控制也不错，日常使用时几乎听不到风扇声。

## 六、 适合人群与选购建议

**适合人群：**
- 追求极致性能和可升级性的用户
- 需要强大核显的游戏玩家
- 想搭建All-in-One的极客用户
- 视频创作者（轻度剪辑）

**不适合人群：**
- 预算有限的用户（准系统约2500元）
- 对体积极度敏感的用户
- 需要独显的用户

**选购建议：**
- 强烈建议购买**准系统**，自行购买内存和硬盘，性价比更高
- 内存建议32GB DDR5-5200MHz以上
- SSD建议选择PCIe 4.0高速盘（如三星990 Pro）
- 首批建议购买8700G版本，未来可升级到zen5架构


## 七、 总结

Minisforum MS-A1是一款**划时代的产品**。它首次将AM5插槽带入迷你主机，让用户可以像台式机一样自由升级CPU。这意味着你买一台主机，可以用十年甚至更久——未来zen5、zen6架构的CPU都能兼容。

**核显性能**也是其最大亮点——Radeon 780M是目前最强的集成显卡，可以流畅运行大多数主流游戏。对于不想买独立显卡但又希望有游戏性能的用户来说，这是目前最好的选择。
