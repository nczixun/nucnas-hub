# 2026年AMD 8845HS迷你主机推荐：高性能mini PC选购指南

## 为什么选择AMD 8845HS

AMD Ryzen 7 8845HS是AMD在2024年初推出的移动级处理器，采用Zen 4架构和RDNA 3核显。凭借8核16线程的强大规格和出色的能效比，成为高性能迷你主机的首选CPU。

## 8845HS核心规格

| 参数 | 规格 |
|------|------|
| 核心架构 | Zen 4 |
| 核心/线程 | 8C/16T |
| 基础频率 | 3.8GHz |
| 睿频 | 5.1GHz |
| TDP | 45W (可配置35-54W) |
| 核显 | Radeon 780M (12CU) |
| 核显频率 | 2.5GHz |
| 三级缓存 | 16MB |
| 最大内存 | DDR5-5600 64GB |

## 性能表现

### CPU性能

8845HS的Cinebench R23多核得分可达15800分，单核得分约1850分。性能接近桌面级i5-12400，满足重度办公、轻度渲染等需求。

### GPU性能

Radeon 780M核显性能接近GTX 1050 Ti，可以：
- 流畅运行《英雄联盟》《CS2》
- 支持轻度3D建模
- 加速视频编码

### AI性能

8845HS内置NPU（神经网络处理器），AI算力达16 TOPS，支持Windows Studio Effects等AI功能。
