# Minisforum HX100G 评测：RX6650M 独显 + 液金散热，游戏性能实测

最近上手了 Minisforum HX100G，这台迷你主机最大亮点是**搭载 AMD Ryzen 7 7840HS + RX 6650M 独立显卡**，在 1L 体积内实现了不错的游戏性能。

## 核心配置

| 参数 | 规格 |
|------|------|
| CPU | AMD Ryzen 7 7840HS (8核16线程) |
| 显卡 | AMD RX 6650M (8GB GDDR6) |
| 内存 | DDR5 5600 32GB |
| 存储 | NVMe 1TB |

## 性能实测数据

**3DMark 跑分：**
- Time Spy：8200 分
- Fire Strike：15800 分
- Port Royal：3800 分

**游戏实测（1080P 高画质）：**
- 《黑神话：悟空》：45 FPS
- 《赛博朋克 2077》：52 FPS
- 《古墓丽影：暗影》：78 FPS

**散热表现：**
HX100G 采用**液金导热 + 双风扇**设计。连续烤机 30 分钟，CPU 温度 85℃，GPU 温度 78℃。风扇噪音略大，满载约 45dB。

## 优缺点总结

**优点：**
- 独显性能强，1080P 游戏无压力
- 液金散热效率高
- 接口丰富：USB4、HDMI 2.1、DP 2.0

**缺点：**
- 满载噪音偏大
- 电源适配器较大
- 价格约 4500 元，性价比一般

适合人群：追求迷你体积 + 游戏性能的用户。如果预算有限，可以考虑核显机型如 SER8。
