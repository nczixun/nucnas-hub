# 铭凡UN100D评测：Intel N100入门NAS神器还是鸡肋？

铭凡UN100D是一款搭载Intel N100处理器的迷你主机，定位入门级市场。有人说它是性价比NAS神器，有人说是鸡肋。实测告诉你真相。

## 外观与接口

UN100D采用紧凑型设计，机身小巧：

| 尺寸 | 130×127×50mm |
|------|--------------|
| 重量 | 约400g |
| 材质 | 金属+塑料 |

接口配置：
- 2×USB 3.2 Gen2
- 2×USB 2.0
- 1×HDMI 2.0
- 1×DP 1.4
- 1×千兆网口
- 1×DC电源

接口不算丰富，但对于NAS场景够用。

## 性能测试

### CPU性能

N100是Alder Lake-N架构，4核心4线程：

| 测试项目 | 分数 |
|----------|------|
| Geekbench 6 单核 | 1180 |
| Geekbench 6 多核 | 3450 |
| Cinebench R23 单核 | 920 |
| Cinebench R23 多核 | 2980 |

### 功耗表现

| 状态 | 功耗 |
|------|------|
| 待机 | 6W |
| 满载 | 15W |
| 关机 | 1W |

功耗控制非常优秀。

### 温度表现

| 状态 | 温度 |
|------|------|
| 待机 | 35℃ |
| 满载 | 65℃ |

散热无压力。

## NAS场景测试

### 轻量级服务

| 服务 | 能否运行 | 体验 |
|------|----------|------|
| Alist | ✅ 流畅 | 良好 |
| Jellyfin 视频转码 | ⚠️ 吃力 | 一般 |
| Home Assistant | ✅ 流畅 | 良好 |
| Docker | ✅ 可以 | 轻量服务OK |

### 文件存储

N100处理小文件读写完全没有问题，USB3.2外接硬盘盒速度正常。但Esata接口缺失略有遗憾。

## 做NAS的优缺点

### 优点
1. 功耗极低，24小时开机不心疼
2. 体积小巧，不占空间
3. 价格便宜，入门首选
4. 支持Docker，可玩性强

### 缺点
1. 核显解码能力有限，4K转码吃力
2. 只有一个网口，无法做链路聚合
3. 扩展性一般，内存不可升级
4. 无UPS接口，意外断电风险

## 适合人群

**推荐购买：**
- 第一次玩NAS的新手
- 只做文件存储+轻量Docker
- 预算紧张的用户
- 需要一台24小时下载机

**不建议购买：**
- 需要4K转码看片
- 需要双网口做软路由
- 追求数据安全需要双盘RAID

## 总结

铭凡UN100D是入门级NAS的好选择，但不要期望太高。它适合：
- 体验NAS功能
- 做下载机
- 运行轻量Docker服务

如果你需要更强的性能和更多的功能，建议加钱上更专业的NAS设备。N100做NAS：够用，但仅仅是够用。
