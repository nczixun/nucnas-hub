# 绿联 DX4600 Pro 评测

绿联 DX4600 Pro 是绿联推出的旗舰级 NAS，搭载 Intel N5095 处理器。

## 核心配置

| 组件 | 规格 |
|------|------|
| 处理器 | Intel N5095 |
| 核心数 | 4 核 4 线程 |
| 基础频率 | 2.0 GHz |
| 睿频 | 2.9 GHz |
| 内存 | 8GB DDR4 (可扩展) |
| 盘位 | 4× 3.5" + 2× M.2 |

## 性能测试

| 测试 | 分数 |
|------|------|
| Geekbench 5 单核 | 850 |
| Geekbench 5 多核 | 2800 |

## 接口

- 2× 2.5G 网口
- 2× USB 3.2
- 1× USB-C
- 1× HDMI

## 优点

- 性价比高
- 系统易用
- 硬件配置强

## 缺点

- Docker 功能有限
- 进阶功能需付费

## 总结

绿联 DX4600 Pro 是入门级 NAS 的绝佳选择。
