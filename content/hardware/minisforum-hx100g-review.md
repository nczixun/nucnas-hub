# Minisforum HX100G评测：RX6650M独显+液金散热，游戏小钢炮来了

迷你主机也能玩3A游戏？**Minisforum HX100G**给出了答案。这款机器搭载AMD 7840HS处理器和RX6650M独立显卡，配置相当硬核，是目前性能最强的迷你主机之一。

## 核心配置

| 配置项 | 参数 |
|--------|------|
| 处理器 | AMD Ryzen 7 7840HS (8核16线程) |
| 显卡 | AMD Radeon RX6650M (8GB GDDR6) |
| 内存 | DDR5 5600MHz (双通道) |
| 存储 | M.2 2280 NVMe SSD × 2 |
| 网口 | 2.5G × 2 |
| 功耗 | 整机约180W |

## 外观设计

HX100G采用了**全金属机身**，质感非常不错。体积约为2.5L，比普通办公迷你主机稍大，但比传统游戏主机小很多。

机身接口：
- HDMI 2.1 × 1
- DP × 1
- USB 4 × 1 (40Gbps)
- USB 3.2 × 4
- 2.5G网口 × 2
- SD卡槽
- OCuLink (外接显卡坞)

接口相当丰富，特别是OCuLink接口可以外接桌面显卡，这在迷你主机上很少见。

### 散热与噪音

HX100G采用了**液金散热**，导热效率比传统硅脂更高。满载时：
- CPU温度：约75°C
- GPU温度：约70°C
- 噪音：约40dB (游戏模式)
