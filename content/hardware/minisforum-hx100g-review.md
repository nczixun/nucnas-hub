# Minisforum HX100G评测：RX6650M独显加持的硬核迷你主机

对于追求极致性能的迷你主机用户来说，核显性能往往不够用。之前测评过不少AMD 8845HS的迷你主机，核显性能虽然强劲，但面对3A游戏时还是略显吃力。今天给大家带来一款真正能打的——**Minisforum HX100G**，搭载AMD R7-5800H处理器和RX6650M独立显卡。

## 核心配置

| 配置项 | 参数 |
|--------|------|
| 处理器 | AMD R7-5800H (8核16线程) |
| 显卡 | AMD RX6650M (8GB显存) |
| 内存 | DDR4 3200MHz (支持双通道) |
| 存储 | M.2 NVMe SSD + 2.5寸硬盘位 |
| 功耗 | 整机约180W |

## 外观设计

HX100G采用了**全金属机身**，质感相当不错。体积约为2.8L，在独显迷你主机中算是比较紧凑的。机身采用深灰色配色，顶部有散热出风口，整体设计比较低调。

机身接口丰富：
- HDMI 2.1 × 2
- DP 1.4 × 2
- USB 3.2 × 4
- 2.5G网口 × 2
- USB-C (全功能)
- 3.5mm音频接口

## 性能实测

### CPU性能

R7-5800H基于Zen 3架构，单核性能在2026年依然够用，多核性能也相当不错。日常办公、编程完全不在话下。

### 显卡性能

RX6650M对标NVIDIA RTX 3050Ti，在1080P分辨率下可以流畅运行大部分3A游戏：

| 游戏 | 画质 | 帧率 |
|------|------|------|
| 赛博朋克2077 | 中等 | 45fps |
| 刺客信条英灵殿 | 高 | 58fps |
| 古墓丽影暗影 | 高 | 62fps |
| 永劫无间 | 高 | 70fps |

### 散热表现

HX100G采用了**液金散热**，散热效果确实不错。室温25℃下，双烤30分钟CPU温度约85℃，GPU温度约78℃。风扇噪音中等，游戏时约为45dB。

**Minisforum HX100G**是目前市面上性价比最高的独显迷你主机之一。RX6650M的性能足以应对大多数3A游戏，液金散热也保证了长时间运行的稳定性。售价约为**4500元**，性价比相当不错。
