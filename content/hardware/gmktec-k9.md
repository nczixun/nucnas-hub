# 性能党的新选择——GMKtec K9 评测：双风扇散热让Ultra 125H火力全开

**测评日期：** 2026年3月1日  
**测评型号：** GMKtec K9（Intel Core Ultra 5 125H / 32GB+1TB版）


## 一、 前言

在迷你主机市场，**GMKtec**（极摩客）是一个相对年轻的品牌，但凭借**高性价比和激进定价**，他们迅速在北美和欧洲市场站稳了脚跟。今天我们要评测的**K9**，是GMKtec旗下首款搭载**Intel Core Ultra 5 125H**的机型，主打**双风扇散热**和**雷电4接口**。

这颗Ultra 5 125H处理器，是Intel Meteor Lake架构的中端主力，兼顾性能和功耗。那么K9能否发挥它的全部实力？让我们往下看。


## 二、 外观与设计：实用主义，散热优先

**体积与设计**
K9的机身尺寸为**160×120×50mm**，体积约**1.0升**，比普通NUC略大，但依然非常紧凑。外观设计比较**务实**——全黑塑料机身，顶部采用**菱形纹路设计**，兼顾防滑和美观。

**双风扇散热：这是最大的特色**
K9内部配备**两个主动散热风扇**：
- CPU散热风扇（40mm）
- 机箱风道风扇（50mm）

这种**双风扇设计**在迷你主机中非常罕见，主要目的是确保Ultra 5 125H能够**长时间稳定运行在最高睿频**。

**接口配置**

| 位置 | 接口类型 | 规格 |
| :--- | :--- | :--- |
| **前置** | 电源键、USB4（40Gbps）×2、3.5mm音频 | 高速传输 |
| **后置** | 2×HDMI 2.0、2.5G RJ45、USB 3.2 Gen2×3、USB 2.0×2、DC 19V | 丰富接口 |


## 三、 硬件配置：Ultra 5 125H中端主力

| 组件 | 规格 |
| :--- | :--- |
| **处理器** | Intel Core Ultra 5 125H（4P+8E+2LPE，14核18线程） |
| **集成显卡** | Intel Arc Xe-LPG（7核，2.2GHz） |
| **NPU** | Intel AI Boost（NPU算力11 TOPS） |
| **内存** | 2×SO-DIMM，支持最高96GB DDR5-5600MHz |
| **存储** | 2×M.2 2280 NVMe插槽（均支持PCIe 4.0×4） |
| **网络** | 2.5G RJ45（Realtek RTL8125） |

**性能表现**
- CPU-Z单核得分：720分，多核得分：5800分
- Arc Xe核显：Time Spy得分约2200分
- NPU算力：11 TOPS，支持本地AI任务


## 四、 散热测试：双风扇有多强？

**温度测试**
- 待机温度：32°C
- 单烤CPU 10分钟：65°C（风扇转速35%，几乎静音）
- 单烤CPU 30分钟：68°C（风扇转速45%，噪音35dB）
- 双烤CPU+GPU：75°C（风扇转速60%，噪音42dB）

**散热评价**
双风扇设计的效果非常明显——即使长时间满载，K9也能将温度控制在**75°C以内**，远低于Intel建议的100°C上限。这意味着CPU可以**持续保持睿频**，不会因为过热而降频。

**噪音控制**
- 待机：完全静音（风扇停转）
- 轻度负载：30dB，几乎听不见
- 满载：42dB，略高于平均水平


## 五、 雷电4：外接设备的终极方案

K9配备**2个雷电4接口**，这是同价位产品中非常罕见的配置。

**雷电4的玩法：**
1. **外接显卡坞** - 连接eGPU，获得接近桌面级的游戏性能
2. **高速存储** - 外接NVMe SSD，速度可达3000MB/s
3. **多屏输出** - 菊花链连接多台显示器
4. **一线通** - 同时充电、数据传输、视频输出
