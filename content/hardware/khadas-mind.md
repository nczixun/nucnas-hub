# 迷你主机的“模块化革命”——Khadas Mind 评测：随身携带的性能怪兽

**测评日期：** 2026年3月1日  
**测评型号：** Khadas Mind（Intel Core Ultra 7 155H / 32GB+1TB版）


## 一、 前言

提到**Khadas**，熟悉树莓派和ARM开发板的用户一定不陌生——他们此前推出的Edge系列开发板和VIM系列单板电脑，在极客圈中口碑极佳。如今，Khadas推出了旗下首款**Intel平台迷你主机**——**Mind**，主打**模块化设计和内置电池**，定位于“移动工作站”。

这台仅有**0.8升**的小主机，不仅可以随身携带，还能通过**雷电4模块**外接显卡坞，甚至可以叠叠乐一样堆叠扩展。这究竟是噱头还是创新？让我们一探究竟。


## 二、 外观与设计：精致的模块化小盒子

**体积与设计**
Mind的机身尺寸为**146×105×20mm**，体积仅**0.8升**，重量约**450g**。这是什么概念？比一台iPad还小，可以轻松放入口袋或背包。机身采用**CNC铝合金材质**，阳极氧化工艺，质感非常高级，有**深空灰和月光银**两种配色。

**模块化设计：这是最大的亮点**

Mind采用了独特的**模块化金手指接口**设计，机身底部有**40-pin连接器**，可以连接多种扩展模块：

1. **Mind Graphics（显卡模块）** - 外接RX 7600M XT显卡坞
2. **Mind Drive（存储模块）** - 增加2TB NVMe存储
3. **Mind Network（网卡模块）** - 增加Wi-Fi 6E和2.5G网口
4. **Mind Power（电池模块）** - 内置4000mAh电池，提供约2小时续航

这种“叠叠乐”设计，让Mind的可玩性大大提高。


## 三、 硬件配置：Intel Ultra 7 155H加持

| 组件 | 规格 |
| :--- | :--- |
| **处理器** | Intel Core Ultra 7 155H（6P+8E+2LPE，16核22线程） |
| **集成显卡** | Intel Arc Xe-LPG（8核，2.25GHz） |
| **NPU** | Intel AI Boost（NPU算力11 TOPS） |
| **内存** | 板载32GB LPDDR5-7467MHz（不可升级） |
| **存储** | 1×M.2 2280 NVMe插槽（支持PCIe 4.0×4） |
| **网络** | Wi-Fi 6E + 蓝牙5.3 |
| **电池** | 机身无电池（需搭配Mind Power模块） |

**性能表现**
- CPU-Z单核得分：780分，多核得分：6800分
- Arc Xe核显：Time Spy得分约2500分，接近GTX 1650
- **NPU加持**：支持本地AI推理，Windows Studio Effects等AI功能


## 四、 接口与扩展：雷电4是亮点

| 接口类型 | 数量 | 规格 |
| :--- | :--- | :--- |
| **雷电4** | 2 | 40Gbps，支持外接显卡坞 |
| **USB-C** | 1 | 10Gbps，支持PD 100W供电 |
| **USB-A** | 2 | USB 3.2 Gen2 |
| **HDMI 2.1** | 1 | 8K@60Hz / 4K@120Hz |
| **3.5mm音频** | 1 | 支持Hi-Fi |
| **TF卡槽** | 1 | 读卡器 |

**雷电4的玩法：**
- 外接显卡坞（推荐Mantle eGFX）
- 外接显示器（菊花链）
- 高速数据传输（40Gbps）


## 五、 内置电池模块：真正的移动工作站

**Mind Power模块**
这是Mind的核心卖点之一。Mind Power模块内置**4000mAh锂电池**，通过底部金手指与主机连接。

**续航表现：**
- 办公续航：约2小时
- 视频播放：约1.5小时
- 充电：支持65W PD快充，30分钟充至50%

**使用场景：**
- 临时外出办公，无需携带电源适配器
- 会议演示，无需找插座
- 露营、旅途中的娱乐工作站

3. **创意工作者** - 需要高性能但不想背游戏本
4. **多设备用户** - 家里一台、公司一台，通过模块实现无缝切换


## 七、 适合人群与选购建议

**适合人群：**
- 追求极致便携性的专业用户
- 喜欢模块化设计的极客
- 需要随时随地办公的商务人士

**不适合人群：**
- 预算有限的用户（整机约5000元）
- 需要高性能独显的用户

**选购建议：**
- 建议购买**套装版**（主机+Power模块+Graphics模块），体验最完整
- 内存为板载，建议直接选32GB版本
- 如果不需要显卡坞，可以只买主机+Power模块
