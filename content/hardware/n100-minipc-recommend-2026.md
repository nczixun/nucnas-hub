## 一、什么是N100？

### 1.1 处理器介绍

Intel N100是Intel第12代Alder Lake-N系列处理器，采用全新的Gracemont架构：

| 参数 | 规格 |
|-----|------|
| 核心数 | 4核心4线程 |
| 基础频率 | 1.8GHz |
| 睿频 | 3.4GHz |
| 缓存 | 6MB L3 |
| 功耗 | 6W TDP |
| 制程 | Intel 7 (10nm) |
| 核显 | 24EU Xe核显 |

### 1.2 性能表现

N100虽然定位入门，但性能并不弱：

| 测试项目 | 性能表现 |
|---------|---------|
| CPU-Z单核 | 约350分 |
| CPU-Z多核 | 约1400分 |
| Cinebench R23单核 | 约800分 |
| Cinebench R23多核 | 约2800分 |
| 核显性能 | 接近GTX 750 |

**简单来说**：N100的性能接近 i3-8100T，日常办公、视频播放完全够用，玩玩英雄联盟也没问题！
