## 五、BIOS 设置

### 5.1 进入 BIOS

开机时按 **F2** 或 **Del** 键进入 BIOS 设置界面。

### 5.2 关键设置项

| 设置项 | 推荐值 | 说明 |
|--------|--------|------|
| Boot Order | U 盘优先 | 设置从 U 盘启动 |
| Secure Boot | Disabled | 关闭安全启动 |
| VT-x/VT-d | Enabled | 开启虚拟化 |
| CSM/Legacy | Disabled | 建议使用 UEFI |

### 5.3 避坑指南 ⚠️

> **坑 10**：不同品牌电脑进入 BIOS 的按键不同（F2/Del/F12/Esc）
> 
> **坑 11**：部分主板需要刷新 BIOS 才能支持新 CPU
> 
> **坑 12**：笔记本注意关闭 **Fast Boot** 选项

## 十三、常见问题 FAQ

### Q1: 安装后无法启动？

检查 U 盘是否正确识别，尝试重新制作启动盘。

### Q2: 客户端无法连接？

检查网络连接，确保在同一局域网。使用 IP 或 GEAK ID 连接。

### Q3: Docker 运行缓慢？

增加内存，或关闭不必要的容器。

### Q4: 影视库无法刮削？

检查网络连接，GEAK AI 可能需要较长时间下载。

### Q5: 如何开启 SSH？

⚠️ 官方默认不开放 SSH，比较遗憾。可以通过 Docker 容器安装 SSH 服务。
