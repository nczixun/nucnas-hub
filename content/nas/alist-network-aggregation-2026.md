## 三、本地部署教程

### 3.1 环境要求

- NAS设备（群晖、威联通、绿联、极空间等）
- 或者任何支持Docker的Linux服务器
- 小主机/NUC也可以完美运行

### 3.2 Docker安装（推荐）

```bash
# 创建配置文件目录
mkdir -p /opt/alist/data

# 启动容器
docker run -d \
  --restart=always \
  -v /opt/alist/data:/opt/alist/data \
  -p 5244:5244 \
  --name alist \
  alistv3/alist:latest
```

### 3.3 初始化配置

1. 浏览器访问 `http://你的IP:5244`
2. 默认管理员账号：`admin`
3. 默认密码：`alist`
4. **首次登录后务必修改密码！**

### 3.4 添加网盘

1. 登录管理后台
2. 点击「存储」→ 「添加」
3. 选择你要添加的网盘类型
4. 根据提示填写对应的Token/Refresh Token
5. 保存即可

> **提示**：不同网盘的获取Token方式不同，建议查阅Alist官方文档获取最新教程。

## 五、常见问题

### Q1: Alist安全吗？
A：Alist本身是开源的，代码可查。部署在本地NAS上，所有数据都在自己手里，非常安全。

### Q2: 会封号吗？
A：Alist只是调用网盘的官方API，理论上存在风控风险。建议使用小号或者非主力号进行操作。

### Q3: 速度怎么样？
A：速度取决于你的网络环境和网盘的官方限速，Alist本身不提供加速服务。

### Q4: 手机上怎么用？
A：iOS和Android都有官方客户端，可以直接在应用商店搜索「Alist」下载。
