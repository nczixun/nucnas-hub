# 极空间Z4Pro性能版深度评测：Intel N305加持，这可能是最强家用NAS

**测评日期：** 2026年3月

## 一、 前言

极空间Z4Pro性能版是极空间最新推出的旗舰级NAS，搭载Intel N305处理器，定位高端家庭用户和小型工作室。相比上代Z4S，Z4Pro在性能和扩展性上都有显著提升。

这款产品究竟表现如何？是否值得升级？让我们详细评测。

## 二、 外观与设计

### 2.1 机身设计

| 参数 | 值 |
| :--- | :--- |
| 尺寸 | 227×225×180mm |
| 材质 | 航空级铝合金 |
| 颜色 | 钛金灰 |
| 硬盘位 | 4×3.5"+2×M.2 |

### 2.2 接口配置

| 位置 | 接口 |
| :--- | :--- |
| **前置** | USB-C 10Gbps、USB-A 10Gbps、电源键 |
| **后置** | 2×2.5G网口、USB 2.0、HDMI 2.0、DC电源 |

### 2.3 散热设计

- 贯穿式风道
- 双14025风扇（支持智能调速）
- CPU独立散热片

## 三、 核心硬件

### 3.1 处理器

| 参数 | Intel N305 |
| :--- | :--- |
| 核心/线程 | 8C/8T |
| 频率 | 3.8GHz |
| TDP | 15W |
| 核显 | UHD Graphics (32EU) |

### 3.2 内存与存储

- **内存**：8GB DDR5（可扩至32GB）
- **存储**：
  - 4×3.5" SATA硬盘位
  - 2×M.2 NVMe插槽（可做缓存或存储）

### 3.3 网络

- 2×2.5G网口
- 支持链路聚合（最高5G）
- 支持SMB多通道

## 四、 系统体验

### 4.1 极空间OS

极空间的系统以"易用"著称：
- ⏱️ 3分钟快速部署
- 📱 全平台客户端支持
- 🖥️ 类Windows文件管理
- 🌐 内网/外网穿透（免费）

### 4.2 核心功能

| 功能 | 评价 |
| :--- | :--- |
| **影视墙** | 4K解码流畅，刮削准确 |
| **AI相册** | 人脸识别、场景分类、回忆功能 |
| **下载** | 支持BT/PT/磁力/网盘 |
| **Docker** | 支持，图形化界面友好 |
| **虚拟机** | 支持（需16GB+内存） |

## 五、 性能实测

### 5.1 文件读写（SMB）

| 测试环境 | 读取 | 写入 |
| :--- | :--- | :--- |
| 单2.5G | 283MB/s | 245MB/s |
| 聚合5G | 520MB/s | 480MB/s |

### 5.2 4K视频转码

| 格式 | 硬件转码 | 软件转码 |
| :--- | :--- | :--- |
| 4K H.265 → 1080P | 60fps | 25fps |
| 8K H.265 → 1080P | 45fps | 12fps |

| 容器 | 启动时间 | 内存占用 |

| 状态 | 功耗 |
| :--- | :--- |

## 六、 特色功能详解

### 6.1 极影视

- 支持Plex/Emby/Jellyfin核心功能
- 自动刮削海报、简介
- 支持蓝光原盘播放
- HDMI直连电视（输出4K@60fps）

### 6.2 极相册

- AI人脸识别
- 场景自动分类（风景、美食等）
- 宝宝相册功能
- 支持苹果Live Photo

### 6.3 远程访问

- 免费P2P穿透
- 无需公网IP
- 速度尚可（非会员限速）

## 七、 总结

**优点：**
- ✅ Intel N305性能强劲
- ✅ 系统易用，小白也能上手
- ✅ Docker/虚拟机支持
- ✅ 影视墙和AI相册体验优秀
- ✅ HDMI直连，客厅NAS首选

**缺点：**
- ❌ 虚拟机性能有限
- ❌ 外网访问非会员限速
- ❌ 价格偏高

**购买建议：**
- **推荐升级**：Z4S及更老用户
- **推荐新人**：追求易用性的家庭用户
- **可选替代**：绿联DH4600、群晖DS920+
