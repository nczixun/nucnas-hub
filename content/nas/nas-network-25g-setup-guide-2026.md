# NAS 网络设置：如何跑满 2.5G 带宽？

路由器、交换机、网线，有一个短板速度就上不去。

**硬件要求：**

- NAS 至少配备 **2.5G 网口**
- 电脑配备 2.5G 网口或 USB 网卡
- 路由器/交换机支持 2.5G
- 至少 **Cat 5e 网线**（推荐 Cat 6）

**实测速度：**

| 场景 | 速度 |
| :--- | :--- |
| 单盘读取 | 180 MB/s |
| 单盘写入 | 160 MB/s |
| RAID 0 读取 | 280 MB/s |
| RAID 0 写入 | 250 MB/s |

**设置步骤：**

1. 确认 NAS 网口速率设为 **2500M Full Duplex**
2. 路由器开启 **链路聚合**（可选）
3. Windows 网卡高级设置：**关闭节能以太网**
4. 用 iPerf3 测试内网速度

速度上不去？先检查网线是不是**千兆头**，很多廉价网线只做了 4 芯。
