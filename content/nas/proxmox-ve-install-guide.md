# Proxmox VE 虚拟机安装教程：小白也能学会

## 什么是 PVE？

Proxmox VE（以下简称 PVE）是一款开源虚拟化平台，基于 Debian 开发，支持 KVM 虚拟化和 LXC 容器。简单理解：它让你在一台硬件上同时运行多个"电脑"，省钱又省心。

## 准备工作

- **硬件要求**：双核 CPU、4GB+ 内存、32GB+ SSD
- **安装介质**：8GB 以上 U 盘
- **下载镜像**：访问 [proxmox.com](https://www.proxmox.com) 下载 ISO

## 详细安装步骤

### 第一步：制作启动盘

下载 Ventoy 或 Rufus，将 PVE ISO 写入 U 盘。

### 第二步：设置 BIOS

开机按 Del/F2 进入 BIOS，确保：
- 开启 **Intel VT-x** 或 **AMD-V** 虚拟化
- 关闭 Secure Boot
- 设置 U 盘启动

### 第三步：开始安装

1. 选择 "Install Proxmox VE"
2. 同意许可协议
3. 选择目标磁盘（建议用 SSD）
4. 设置国家、时区、键盘布局
5. 配置网络（建议用静态 IP）：
   - IP 地址：192.168.x.x
   - 子网掩码：255.255.255.0
   - 网关：路由器 IP
   - DNS：8.8.8.8
6. 设置管理员密码和邮箱
7. 确认配置，点击 "Install"

安装耗时约 10-15 分钟。

### 第四步：初始化配置

1. 浏览器访问 `https://你的IP:8006`
2. 用户名：root，密码：刚才设置的密码
3. 首次进入会弹出订阅提示，选择 "OK" 跳过

## 快速上手：创建第一个虚拟机

1. 点击右上角 "创建 VM"
2. 名称：随便填，如 "Ubuntu-Test"
3. ISO 镜像：上传系统 ISO 或选择光盘驱动
4. 磁盘大小：建议 30GB 以上
5. CPU 核心：分配 2 核
6. 内存：分配 4GB
7. 网络：默认 virtio 半虚拟化
8. 点击 "完成"，然后点击 "启动"

## 常见问题

**Q：PVE 和 ESXi 有什么区别？**
A：PVE 免费开源，ESXi 免费版功能受限。PVE 对中文用户更友好。

**Q：需要独立显卡吗？**
A：不需要，核显足够日常虚拟机使用。

**Q：数据盘怎么挂载？**
A：进入 "数据中心" → "存储" → "添加" → "磁盘"，选择数据盘即可。

---

PVE 虚拟化是 NAS 进阶的必经之路。安装完成后，你可以同时运行 OpenWrt 旁路由、Home Assistant、Ubuntu Server 等多个系统，一台机器搞定所有需求。
