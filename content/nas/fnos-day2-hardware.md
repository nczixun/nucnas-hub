# Day 2: 飞牛OS硬件选购指南

## 入门方案：J1900/J1800 工控机

**价格：100-200元（闲鱼）**

这是最经济的入门方案，适合轻量级使用：

| 规格 | 参数 |
|------|------|
| CPU | Intel J1900 (4核4线程) |
| 内存 | 最大8GB DDR3L |
| 功耗 | 10W TDP |
| 接口 | 2×SATA3, 千兆网口, USB 3.0 |

**优点：** 价格便宜，省电
**缺点：** 性能较弱，不适合Docker大量部署

## 进阶方案：联想M920Q

**价格：600-1000元（二手）**

性价比极高的进阶选择：

| 组件 | 推荐配置 |
|------|----------|
| 主机 | 联想M920Q |
| CPU | i3 8100T (35W) |
| 内存 | 16GB (8GB×2) |
| 存储 | NVMe SSD + 机械盘 |
| 电源 | 90W |

**优点：** 性能强劲，核显支持转码，体积小巧
**缺点：** 需要自行购买配件

## 高性能方案：全新组装

| 配置 | 推荐 |
|------|------|
| CPU | Intel N100 / i3-12100 |
| 主板 | ITX迷你主板 |
| 内存 | 16-32GB |
| 机箱 | ITX机箱 + 电源套装 |

## 硬盘选择

### 系统盘
- **推荐：** NVMe SSD 128GB+
- **用途：** 安装系统，运行Docker

### 数据盘
- **推荐：** 西部数据红盘/希捷酷狼
- **容量：** 根据需求选择（4TB起）

## 注意事项

1. ⚠️ 核实网卡驱动：部分机器需要额外驱动
2. ⚠️ 电源功率：确保供电充足
3. ⚠️ 机箱空间：注意兼容性和散热
