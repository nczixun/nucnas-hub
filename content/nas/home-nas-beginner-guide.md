## 一、为什么要搭建家庭私有云？

你是否遇到过这些问题：
- 🔴 网盘下载速度只有几十 KB/s
- 🔴 重要文件被和谐无法访问

- 🔴 隐私照片存在别人服务器不放心

**家庭私有云** 完美解决以上所有问题！


## 二、最简方案：闲置电脑 + Windows

### 准备工作
- ✅ 一台闲置的 Windows 电脑（台式机或笔记本）
- ✅ 路由器（需要开启 SMB/CIFS 服务）
- ✅ 一个闲置硬盘（或者 NAS 专用硬盘）

### 步骤 1：开启 Windows 文件共享

1. 按 `Win + R`，输入 `control` 打开控制面板
2. 选择 "网络和共享中心" → "更改高级共享设置"
3. 开启 **"启用文件和打印机共享"**
4. 找到要共享的文件夹 → 右键 → 属性 → 共享 → 添加 "Everyone"

### 步骤 2：手机端设置

**iOS 用户：**
1. 下载 "文件" App
2. 点击 "连接服务器"
3. 输入 `smb://你的电脑IP`
4. 选择刚才共享的文件夹

**安卓用户：**
1. 下载 "ES文件浏览器" 或 "Solid Explorer"
2. 打开 "网络" → "局域网"
3. 自动发现共享电脑

### 效果展示

| 功能 | 效果 |
| :--- | :--- |
| 手机备份照片 | 自动同步到电脑文件夹 |
| 速度 | 取决于局域网（通常 50-100MB/s） |
| 容量 | 取决于硬盘大小（最大 16TB+） |
