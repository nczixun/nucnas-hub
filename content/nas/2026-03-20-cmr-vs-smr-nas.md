# NAS硬盘CMR和SMR区别：选购必看，太多人踩坑了

## 什么是CMR？

CMR (Conventional Magnetic Recording) 传统磁记录，是硬盘最传统的存储技术。

**工作原理**：
- 每个磁道独立存在，互不重叠
- 写入数据时不会影响相邻磁道
- 读写性能稳定可靠

**CMR硬盘特点**：
```
✅ 随机写入性能好
✅ 适合频繁写入
✅ 数据安全性高
✅ 寿命更长
❌ 成本较高
❌ 存储密度有限
```

## 什么是SMR？

SMR (Shingled Magnetic Recording) 叠瓦式磁记录，是一种提高存储密度的技术。

**工作原理**：
- 磁道像屋顶瓦片一样重叠
- 后一条磁道部分覆盖前一条
- 牺牲随机写入能力换取更高密度

**SMR硬盘特点**：
```
✅ 存储密度高
✅ 成本较低
✅ 大容量型号多
❌ 随机写入性能差
❌ 写入放大效应
❌ 不适合NAS使用
❌ 寿命相对较短
```

## 为什么NAS不能用SMR？

### 实际测试对比

| 测试项目 | CMR硬盘 | SMR硬盘 |
|----------|---------|---------|
| 顺序写入 | 180MB/s | 175MB/s |
| 随机写入4K | 2.1MB/s | 0.3MB/s |
| 写入延迟 | 8ms | 200ms+ |
| RAID重建 | 2小时 | 12小时+ |

### SMR在NAS中的致命问题

1. **写入放大**：每次写入都要先读取、修改、重写相邻磁道
2. **RAID重建失败**：硬盘损坏后，RAID重建时大量写入会导致SMR硬盘崩溃
3. **性能骤降**：多盘同时写入时，SMR硬盘会严重掉速
4. **寿命缩短**：频繁写入会加速SMR硬盘老化

### 方法1：查看官方规格

| 品牌 | 官网查询 |

### 方法2：看型号命名规律

**西数 Red 系列**：
- Red Plus = CMR
- Red (新型号) = 可能是SMR

**希捷酷狼系列**：
- IronWolf = CMR
- IronWolf Pro = CMR

**东芝 N 系列**：
- N300 = CMR

### 方法3：看缓存大小

```
256MB缓存 + 8TB以上 = 大概率SMR
64-128MB缓存 = 可能是CMR
```

## 常见问题

### Q：SMR硬盘真的不能用吗？

A：不是不能用，而是不适合NAS。SMR硬盘适合做仓库盘（只读不写或很少写入），不适合7×24小时高负载的NAS环境。

### Q：为什么市面上SMR硬盘越来越多？

A：因为SMR成本低，厂商利润更高。但对消费者来说，长期使用成本反而更高（寿命短、风险大）。

### Q：有没有简单的方法避开SMR？

A：记住这个选购口诀——**"NAS必选CMR，SMR再便宜也不买"**。
