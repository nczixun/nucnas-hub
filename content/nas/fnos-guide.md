# 如何让你的闲置 NUC 在 10 分钟内变身为强大的家庭媒体中心

很多人的抽屉里都吃着灰躺着一台 Intel NUC。它处理日常办公可能略显吃力，但如果作为一台 7x24小时在线的媒体服务器，它的性能绰绰有余。今天，我们就用最简单的方法，让它复活。🛠️ 准备工具硬件： 闲置 NUC 一台（建议 i3 及以上，带至少 8GB 内存）。系统： 建议预装 Ubuntu Server 或任何你熟悉的 Linux 系统（Windows 亦可，但 Linux 更稳定）。核心： CasaOS —— 这是一个极简的开源家庭云系统，能让你像用手机 App 一样管理服务器。⏱️ 变身步骤（倒计时开始！）第一步：一键安装控制面板（用时：3 分钟）连接好你的 NUC 并打开终端，输入下面这行神级命令：安装完成后，在浏览器输入 NUC 的 IP 地址，你会看到一个非常精致的仪表盘界面。第二步：安装核心影音引擎（用时：3 分钟）在 CasaOS 的 App Store 中，你有两个顶级选择：Plex / Jellyfin： 负责整理你的电影仓库，自动刮削封面、演员信息和中文字幕。推荐选 Jellyfin： 因为它是完全免费的，且支持硬件解码，能榨干 NUC 的核显性能。点击“Install”，静待进度条拉满。第三步：建立资源下载站（用时：2 分钟）媒体中心不能没有内容。继续在 App Store 点击安装 qBittorrent。进阶玩法： 如果你追求自动化，可以搜索并安装 iapp 里的“小雅 (Xiaoya) Alist”，它可以直接挂载海量的阿里云盘资源，让你连下载的时间都省了。第四步：挂载硬盘与分享（用时：2 分钟）将存有电影的移动硬盘插在 NUC 的 USB 3.0 接口上。在 CasaOS 的文件管理器里点击“挂载”，然后在 Jellyfin 的后台添加这个媒体库路径。🌟 为什么 NUC 是最佳选择？硬件解码（QSV）： Intel 的核显在处理 $4K$ 视频转码时非常强悍，即使你在外面用手机远程看家里的原盘电影，NUC 也能实时转换成适合手机观看的流畅格式。极低功耗： 待机功耗通常不到 10W，一年不关机也费不了多少电费。颜值正义： 放在电视机柜旁边，像个精致的小盒子，毫无违和感。💡 进阶 Tips内网穿透： 如果你想在公司也能看家里的电影，可以装个 Tailscale 或 Cloudflare Tunnel。海报墙： 记得在 Jellyfin 里开启“硬件加速”，选 Intel QuickSync，播放体验会瞬间起飞。搞定！ 现在你的 NUC 已经不再是电子垃圾，而是一个属于你自己的私有“奈飞”。
