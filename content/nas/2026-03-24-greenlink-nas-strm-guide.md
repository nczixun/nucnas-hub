# 绿联NAS影视中心Strm播放实战：打造私人影院

## 前言

绿联NAS私有云系统近日迎来重大更新，影视中心正式支持Strm文件播放。这意味着用户可以更灵活地管理自己的影视资源，实现真正的私人影院体验。本文将手把手教你如何使用这一功能。

## 什么是Strm文件？

Strm（Stream链接文件）是Kodi等媒体中心常用的文件格式。本质上是一个包含外部媒体链接的文本文件，播放时直接调用外部源进行流式传输。

**Strm的优势**：
- 不占用NAS本地存储空间
- 支持绑定Emby/Jellyfin/Plex等媒体服务器
- 资源更新便捷
- 支持海报墙自动匹配

## 配置步骤

### 第一步：准备媒体文件

1. 在NAS上创建影视文件夹目录
2. 准备电影/电视剧的Strm文件
3. 确保文件命名规范（建议：电影名.年份.分辨率.strm）

### 第二步：开启影视中心Strm支持

1. 登录绿联NAS管理后台
2. 进入「影视中心」设置
3. 开启「Strm文件支持」选项
4. 选择媒体库路径

### 第三步：配置媒体库

```
媒体库路径：/volume1/Media
├── Movies/（电影）
├── TV/（电视剧）
└── Strm/（Strm文件）
```

### 第四步：扫描并匹配海报

1. 点击「扫描媒体库」
2. 选择需要扫描的文件夹
3. 等待海报墙自动匹配
4. 如有未匹配项，可手动搜索修正

## 常见问题

### Q1: Strm播放卡顿怎么办？

检查网络带宽，确保播放设备与NAS之间的网络稳定。建议使用有线连接而非WiFi。

### Q2: 无法识别Strm文件？

确认文件后缀名为.strm，且文件内容格式正确（应为有效的播放链接）。

### Q3: 海报匹配失败？

尝试手动搜索影片名称，或使用TMDb API进行匹配。

## 进阶玩法

### 联合Emby使用

1. 安装Emby Server
2. 配置Emby媒体库路径为Strm文件夹
3. 启用Emby Connect实现远程播放

### 远程访问设置

1. 开启DDNS服务
2. 配置端口转发
3. 使用绿联Cloud实现外网访问

## 总结

绿联NAS影视中心的Strm支持，大大提升了私有云的媒体播放能力。通过本文的教程，你可以轻松打造属于自己的私人影院，享受4K原盘的震撼画质。
