## 一、什么是NAS？

### 1.1 NAS的定义

**NAS = Network Attached Storage（网络附加存储）**

简单来说，NAS就是一台**24小时开机的家庭数据中心**，它可以：

- 📂 集中存储所有文件、照片、视频
- 📱 自动备份手机/电脑文件
- 🎬 打造私人影院4K视频库
- 🔄 私有化PT/BT下载
- 🏠 智能家居中控中心
- ☁️ 运行Docker应用：Home Assistant、Nextcloud等

### 1.2 NAS vs 网盘

| 对比 | NAS | 百度网盘/阿里云盘 |
|------|-----|-----------------|
| 速度 | 局域网内网满速 | 开通会员才快 |
| 容量 | 硬盘自己买，永久 | 会员到期可能清空 |
| 隐私 | 数据全在自家硬盘 | 隐私泄露风险 |
| 费用 | 一次性投入 | 长期会员费 |
| 24小时在线 | ✅ | ❌ 需手动 |

### 1.3 谁需要NAS？

- 📸 摄影爱好者/视频创作者（大量素材存储）
- 👨‍👩‍👧‍👦 家庭用户（照片备份、影音娱乐）
- 🏠 智能家居玩家（Home Assistant）
- 💻 程序员/极客（Docker、代码仓库）
- 🌐 远程办公（随时随地访问文件）

## 二、NAS选购指南

### 2.1 预算分档

| 预算 | 推荐方案 | 适合人群 |
|------|---------|---------|
| 500-1000元 | 蜗牛星际/矿渣NAS | 练手体验 |
| 1000-2000元 | 威联通TS-212P3/群晖DS220+ | 入门家用 |
| 2000-4000元 | 威联通TS-464C/群晖DS920+ | 进阶玩家 |
| 4000元以上 | 群晖DS1821+/威联通TVS-872X | 专业用户 |

### 2.2 品牌选择

- **群晖（Synology）**：系统生态最强，适合新手
- **威联通（QNAP）**：硬件性价比高，功能丰富
- **绿联（UGREEN）**：国产新势力，简单易用
- **极空间**：国产私有云，操作简单
- **DIY**：黑群晖/Unraid，适合折腾

### 2.3 盘位选择

| 盘位 | 存储容量 | 建议人群 |
|------|---------|---------|
| 2盘位 | 最大16TB（8TB×2） | 个人/新手 |
| 4盘位 | 最大32TB（8TB×4） | 家庭用户 |
| 6盘位+ | 32TB+ | 专业用户/工作室 |

**建议**：至少4盘位，方便以后扩展和组RAID。

## 四、基础设置教程

### 4.1 首次开机

1. 装上硬盘，连接网线
2. 开机，等待系统启动
3. 在浏览器输入NAS IP或使用手机App
4. 创建管理员账户
5. 初始化存储池（建议SHR模式）

### 4.2 存储池配置

**推荐配置**：
- 存储池类型：SHR（群晖）或RAID 5
- 文件系统：Btrfs（支持快照）或ext4
- 开启硬盘休眠，延长寿命

### 4.3 基础服务开启

| 服务 | 作用 | 推荐程度 |
|------|-----|---------|
| File Station | 文件管理 | ⭐⭐⭐ |

| Video Station | 视频库 | ⭐⭐⭐ |
| Download Station | 下载 | ⭐⭐⭐ |
| Docker | 应用扩展 | ⭐⭐⭐ |
| Hyper Backup | 备份 | ⭐⭐⭐ |

## 八、常见问题

### Q1: NAS会不会很费电？

A: NAS非常省电，待机状态一般**10-30瓦**，一天不到1度电。群晖和威联通都有智能硬盘休眠功能。

### Q2: 需要组RAID吗？

A: 建议至少组RAID 1（镜像）或SHR，可以保护数据安全。但最重要的还是**定期备份**。

### Q3: NAS会影响网速吗？

A: 不会。NAS使用的是内网带宽，不影响外网。观看4K视频完全没问题。

### Q4: 硬盘怎么选？

A: 推荐**NAS专用盘**（希捷酷狼、西部数据红盘Plus），不推荐普通电脑硬盘或二手盘。
