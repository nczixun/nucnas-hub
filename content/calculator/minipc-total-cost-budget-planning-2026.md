# 迷你主机预算规划：总成本到底要多少钱？

买主机只是开始，配件、存储、网络都是钱。

**主机成本：**

| 级别 | 主机价格 | 典型机型 |
| :--- | :--- | :--- |
| 入门 | 1000-1500 元 | GMKtec G3 |
| 中端 | 2000-3500 元 | Beelink SER8 |
| 高端 | 5000-10000 元 | Minisforum MS-01 |

**配件成本：**

| 配件 | 入门 | 中端 | 高端 |
| :--- | :--- | :--- | :--- |
| 内存 (16GB) | 200 元 | 350 元 | 500 元 |
| 存储 (1TB) | 300 元 | 500 元 | 800 元 |
| 显示器 | 0 元 | 0 元 | 0 元 |
| 键鼠套装 | 150 元 | 300 元 | 500 元 |

**年度成本：**

| 项目 | 费用 |
| :--- | :--- |
| 电费 (每天 8 小时) | 15-50 元 |
| 维修/更换 (预估) | 100 元 |

**总预算建议：**

- 入门办公：1500-2000 元
- 进阶办公：3500-4500 元
- 专业工作站：8000-12000 元

别忘了留 10% 的预算给外设和存储扩展。
